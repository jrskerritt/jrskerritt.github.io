Installing Jest on Windows in Visual Studio 2013

January 7, 2015

I've been doing a lot of work and experimentation with React lately, and I have to say it's pretty great. Naturally, I wanted to write some tests for the components I've created, so I decided to try out Jest.

Of course, being on Windows and using Visual Studio 2013, there were bound to be gotchas. I ran into some minor trouble trying to get Jest installed through npm, but with some research I was able to get things working. This post is mostly to document the command that was able to work for me in the hopes that it could help someone like me in the same situation.

Here it is:

$ npm install jest-cli --save-dev --msvs-version=2013 --python=python2.7

A couple things to note:

  1. --msvs-version=2013 is required because one of the dependencies of Jest was trying to target Visual Studio 2010 on my machine. Not sure why, but there it is.
  2. --python=python2.7 is required because I have Python 3.4 installed and in my PATH, but the installation requires Python 2.7.

And that was it! Now to start writing some React tests.
